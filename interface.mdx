## Build screens in the visual editor

Flowboard screens are assembled in the visual editor with reusable components. You add components from the **Components panel**, arrange them on the **canvas**, and fine-tune them in the **properties panel**.

If you want guidance on which component to use, what each one is best for, and how to set it up in the dashboard, see the [Components library](/components/index).
